# Program description
Run the ramachandran plot of 3 amino-acid for a pre-selected protein (.pdb)

## Script example
In the script 3 amino acids were taken for observation.
- GLY
- ALA
- aa. pre Proline

The protein taken is 1an3.pdb (Human Emobin)


## Usability
It is possible to modify the academic year and change the analyzed protein by changing the variables in the script
and downloading another protein into the data / directory.
